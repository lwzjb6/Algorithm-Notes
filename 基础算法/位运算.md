返回`n`的最后一位`1：lowbit(n) = n & -n`  `-n = n`的原码的反码`+1`

### acwing 801. 二进制中1的个数
求每个数的二进制表示中 `1` 的个数。 
```c++
int num_of_1 (int x) {
    int ans = 0;
    while(x) {
        x -= (x & -x); // lowbit(x)
        ans++;
    }
    return ans;
}
```
```c++
int num_of_1 (int x) {
    int ans = 0;
    while(x) {
        x = x & (x - 1); // 每次把最后一位1消去
        ans++;
    }
    return ans;
}
```
